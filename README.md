# TodoToolKit

> a to do list with the ability to do, edit and delete



![Logo](https://github.com/mahdihagollahi/TodoWithNext/blob/main/src/app/assets/Img/Logo.png)




## Installation

> First, clone the project

```bash
git clone https://github.com/mahdihagollahi/TodoToolKit.git
```

> to run TodoToolKit with npm

```bash
cd TodoToolKit
  npm install
  npm run dev
```
